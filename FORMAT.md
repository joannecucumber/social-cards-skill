# 社群圖產生器 — 使用說明

這個工具把 Markdown 轉成 Instagram carousel 圖片（1080×1350 等尺寸）。貼上 Markdown，即時預覽，下載 PNG。

---

## 文件結構

```markdown
---
tag: 標籤文字
source: 來源出處
handle: @帳號
theme: 暖橘
mode: light
ratio: 4:5
cover: gradient
lang: zh
---

封面內容

---

第二頁內容

---

第三頁內容
```

- 第一個 `---...---` 區塊是設定（frontmatter）
- 之後每個 `---` 分隔一頁
- 第一頁自動變封面，最後一頁自動產生 CTA

---

## Frontmatter 設定

| 欄位 | 說明 | 選項 |
|------|------|------|
| tag | 標籤 pill（封面 + 內容頁左上角） | 自訂文字 |
| source | 頁尾左側來源文字 | 自訂文字 |
| handle | 頁尾右側帳號 | @xxx |
| theme | 色彩主題 | 暖橘 / vibe-reader / 莓果紫 / 森林綠 / 玫瑰金 / 日落粉 / 海洋藍 / 午夜 |
| mode | 內容頁底色 | light / dark |
| ratio | 圖片比例 | 4:5（IG 直式）/ 1:1（正方形）/ 9:16（Story）/ 16:9（橫式） |
| cover | 封面樣式 | gradient（漸層）/ solid（純色） |
| lang | CTA 頁語言 | zh（預設）/ en / ja / ko |
| cta | 結尾 CTA 頁 | auto（預設）/ none |

設過品牌設定（工具右上角「品牌設定」）的欄位，Markdown 不寫會自動帶入。

---

## 語法

### 文字

| 寫法 | 效果 |
|------|------|
| `# 標題` | 大標題（64px） |
| `## 副標題` | 中標題（40px） |
| `### 小標題` | 小標題（34px） |
| `**文字**` | 主題色強調 |
| `<br>` | 標題內換行 |

### 頁面元素

| 寫法 | 效果 |
|------|------|
| `---` | 下一頁 |
| `[標籤名]` | 該頁左上角的標籤 pill |
| `- 項目` | 圓點列表（主題色圓點） |
| `> 引用文字` | 底部金句引用框（大引號 + 主題色文字） |
| `\|> 文字` | 側邊 callout（💡 icon + 主題色左邊線） |
| `***` | 頁內水平分隔線 |

### 特殊版型

| 寫法 | 效果 |
|------|------|
| `① **標題**` + 下一行說明文 | 編號卡片（數字 + 標題 + 說明，有背景色） |
| `{87%}` | 超大數字（160px，主題色） |
| `![](url)` | 首圖（頂部 28% 滿版圖片，放在頁面第一行） |
| 整頁只寫 `# 一句話` | 宣言頁（自動置中放大） |

## 封面

第一個 `---` 之前的內容自動變封面（漸層或純色背景）。

```markdown
# 大標題含**強調字**

副標題說明文字

ENGLISH SUBTITLE
```

- `# 標題`：封面主標題
- 中文段落：副標題
- 全大寫英文段落：自動變成小字英文副標
- 多頁時右下角自動顯示「滑動看更多 →」

---

## 內容頁

每個 `---` 之間是一頁。可以用 `[標籤]` 設定該頁的標籤。

```markdown
---

[觀念]

# 頁面標題

內文段落，可以用**粗體**強調。

> 金句引用。
```

---

## 編號卡片

用 ①②③ 開頭，會自動渲染成卡片式：

```markdown
① **標題一**

說明文字。

② **標題二**

說明文字。

③ **標題三**

說明文字。
```

- 數字用主題色顯示
- 每個項目有圓角背景卡片
- 一頁超過 3 個會自動壓縮字體

---

## 大數字

用 `{數字}` 語法，適合數據型頁面：

```markdown
# {87%}

## 的人在第一步就放棄

說明文字。
```

數字會渲染成 160px 主題色大字。

---

## 首圖

在任何內容頁的第一行加 `![](圖片網址)`，圖片會佔頂部 28% 滿版：

```markdown
---

[案例]

![](https://images.unsplash.com/photo-xxxxxxx?w=1080)

# 標題

內文。
```

- 任何頁面類型都能加首圖
- 圖片來源用公開 URL（Unsplash 加 `?w=1080`）
- 可以跟宣言頁搭配（首圖 + 只有標題）

---

## 宣言頁

當內容頁只有一個 `# 標題`（沒有其他內容），會自動置中放大：

```markdown
---

# 但真正的問題**不是工具**
```

可選加一句金句：

```markdown
---

# 工具變快了，<br>但你的注意力**沒有變多**

> 這才是要解決的事。
```

---

## CTA 結尾頁

預設自動產生，根據 `lang` 欄位切換語言。

- `cta: auto`（預設）：自動產生
- `cta: none`：不要 CTA 頁

自訂 CTA：

```markdown
---

cta

# 你的自訂標題

你的自訂文字
```

或：

```markdown
---

[cta]

# 你的自訂標題
```

---

## 自動壓縮

當頁面內容太多（例如 4 個以上的編號卡片），工具會自動縮小字體和間距，避免內容溢出。

---

## 快捷鍵

- 打字即時預覽（300ms debounce）
- `Cmd+Enter`：立即產生
- 點擊預覽圖上的 ↓ 按鈕：下載單張
- 「全部下載」：依序下載所有圖片
