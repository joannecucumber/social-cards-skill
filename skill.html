Prerequisites

What you need

Bun

JavaScript runtime. Install with:
curl -fsSL https://bun.sh/install | bash

Claude Code

Anthropic's CLI tool. Get it here

Installation

Three steps to set up

1

Clone the repository

git clone https://github.com/joannecucumber/social-cards-skill.git cd social-cards-skill
2

Install dependencies

This downloads Puppeteer + Chromium (~170MB, first time only).

bun install
3

Copy the skill to Claude Code

cp .claude/commands/social-cards.md ~/.claude/commands/
Usage

How to use it

In Claude Code

# Type this in Claude Code: /social-cards # Or with content directly: /social-cards <paste your article here>

Claude will ask you

1
What language? (zh / en / ja / ko)
2
Your social handle (e.g. @yourstudio)
3
Color theme preference (or let AI decide)

Standalone CLI

# Render from a markdown file bun render.ts my-article.md output/ # Render from stdin cat content.md | bun render.ts - output/

Or just use the editor

Don't want to install anything? Use the browser editor directly.
